STEP 1: Setting Up The Basic Navbar HTML

Three graphics are placed in a table layout and each graphic is linked to an appropriate HTML document. This will provide the basics for the navigation. In the sample below, the table border has been set to 1 so you can see how the graphics are set in the table layout.


Now, let's take a look at the HTML for this sample. Notice how indenting has been used to make the HTML source a little easier to read. It's not really necessary to have this indenting in your real HTML pages.

<CENTER>
<TABLE BORDER=1 CELLSPACING=0 CELLPADDING=0>
   <TR>
      <TD>
         <A
            HREF="buckingham1.html"
         >
         <IMG
            SRC="navbar-prev.gif"
            WIDTH=96
            HEIGHT=36
            ALT="PREV"
            BORDER=0
            HSPACE=0></A></TD>
      <TD>
         <A
            HREF="buckingham-home.html"
         >
         <IMG
            SRC="navbar-home.gif"
            WIDTH=302
            HEIGHT=36
            ALT="HOME"
            BORDER=0
            HSPACE=0></A></TD>
      <TD>
         <A
            HREF="buckingham3.html"
         >
         <IMG
            SRC="navbar-next.gif"
            WIDTH=96
            HEIGHT=36
            ALT="NEXT"
            BORDER=0
            HSPACE=0></A></TD>
   </TR>
</TABLE>
</CENTER>
