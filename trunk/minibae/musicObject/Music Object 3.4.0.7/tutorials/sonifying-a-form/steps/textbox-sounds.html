STEP 6: Adding the Textbox Sounds

For textboxes we can make use of the ONFOCUS and ONBLUR events. To give end-users the feeling of typing an entry we will start the Groovoid "Background-SimpleTyping" playing whenever a user focuses a textbox, and we will play the Groovoid "UI-CarriageReturn" whenever the user finishes typing in an entry and moves on to another form element. In order to sonify the ONFOCUS and ONBLUR events for all the form's textboxes and textareas, we have added the startTypingSound function to play the Groovoid "Background-SimpleTyping" when the user focuses a textbox or textarea, and we have added the stopTypingSound function to play the Groovoid "UI-CarriageReturn" when the user moves away from a textbox or textarea. Because we are playing the Groovoid "UI-CarriageReturn" through the same "formSounds" Music Object instance that starts playing the looping "Background-SimpleTyping" Groovoid, this has the effect of also stopping the looped typing sound that plays all the while that a textbox or textarea is focused.

In addition to the functions we have added, we have also added event handlers to all the textboxes and textareas in the form. Textboxes are created using the INPUT tag and specifying the TYPE=text attribute. Textareas are created using the TEXTAREA tag. Both of these tags support ONFOCUS and ONBLUR event handlers. We have added a call to the startTypingSound function to the ONFOCUS event handlers of all textboxes and textareas in the form. And we have added a call to the stopTypingSound function to the ONBLUR event handlers of all textboxes and textareas, as well.
