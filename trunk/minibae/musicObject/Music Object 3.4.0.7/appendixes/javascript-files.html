Music Object JavaScript Files

The following table contains a complete list of all the JavaScript files that are offered as part of the Music Object API, including the core Music Object API, the discrete components of the core Music Object API, the language modules, the patches, and the extensions. This list is provided as a convenience so that you can quickly begin development of new sonified content, by simply coming to this central location and grabbing whichever JavaScript files you will need for the site.

How to Use the Table

To copy a code module into your site, simply right-click (click-and-hold for Mac users) on the filename for the code module (appears in the "File Name (Download Link)" column). Then, from the menu that pops up, select the 'Save Link As...' or 'Save Target As...' option and save the file into an appropriate folder of the local copy of your Web site. Make sure not to change the file's name when you save it. If you want to refresh your memory on how to use a specific code module, the "more info..." link will take you to the reference information for the code module.
