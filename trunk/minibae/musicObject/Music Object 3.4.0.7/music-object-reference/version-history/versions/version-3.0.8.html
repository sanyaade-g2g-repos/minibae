Music Object Version 3.0.8
RELEASE DATE: 1999-03-26

If you are upgrading from Music Object version 2.x and would like to know about the significant changes that have occurred between versions 2.x and 3.0.x, please consult the version history for Music Object Version 3.0.0.

What's New In This Version

BUG FIX - Unsupported Unix Platforms
A bug was found in the Music Object code which would cause sonified pages to produce JavaScript errors when viewed on some Unix platforms not supported by the Beatnik Player. The desired behavior is for sonified content to be completely silent and not produce any JavaScript errors when viewed by a browser that is not supported by the Beatnik Player. This bug has been fixed.
OPTIMIZATIONS
Some general code optimizations have been made since version 3.0.5, which have shaved a little off the "music-object.js" file size as well as making certain processes a little more efficient.
