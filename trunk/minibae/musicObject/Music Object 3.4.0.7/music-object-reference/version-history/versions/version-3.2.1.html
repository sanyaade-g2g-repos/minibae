Music Object Version 3.2.1
RELEASE DATE: 1999-11-17

If you are upgrading from Music Object version 2.x and would like to know about the significant changes that have occurred between versions 2.x and 3.0.x, please consult the version history for Music Object Version 3.0.0.

What's New In This Version

WORKAROUND - Crash with Internet Explorer 4.0
An issue was discovered with Internet Explorer 4.0 which caused certain content using Music Object 3.2.0 to crash the browser under very specific conditions. The crash bug only affected content where no Beatnik Player instances were visible within a viewable area of the browser window. The Music Object 3.2.1 maintenance release contains a workaround for this Internet Explorer 4.0 issue and completely prevents the crash from occurring.
BUG FIX - Music.installPlayer Not Working in Internet Explorer 4.0
A bug was found with the Music.installPlayer static method which caused it to produce a JavaScript error in Microsoft Internet Explorer 4.0. This bug has been fixed.
