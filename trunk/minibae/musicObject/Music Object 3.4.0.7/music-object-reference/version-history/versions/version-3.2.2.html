Music Object Version 3.2.2
RELEASE DATE: 1999-12-13

If you are upgrading from Music Object version 2.x and would like to know about the significant changes that have occurred between versions 2.x and 3.0.x, please consult the version history for Music Object Version 3.0.0.

What's New In This Version

BUG FIX - onMetaEvent Method Not Working
A bug was found with the onMetaEvent instance method that prevented authors from being able to register meta event handlers. This bug crept in at Music Object 3.1.0 and has now been fixed in the 3.2.2 maintenance release.
BUG FIX - magicEmbed & Music.magicEmbed
A minor bug was found with the magicEmbed instance method and the Music.magicEmbed static method where specifying an empty string for any attribute would cause a player instance to be embedded incorrectly. For example, a statement such as myMusicObject.magicEmbed ('SRC="" WIDTH=2 HEIGHT=2 HIDDEN') would have caused problems because of the empty string specified for the SRC attribute. This bug has been fixed.
