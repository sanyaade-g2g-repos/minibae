About This Patch

If you've been authoring using a version of Music Object before 3.3.0 and would now like to upgrade to the latest version, there may be some features of the older version of Music Object that you were using that are no longer supported in the recent version. Several changes to the API were made between version 3.2.x and version 3.3.0. You should consult the Music Object Version History to find out more about the changes in version 3.3.0 and how they may affect your code.

If you find that your code will, indeed, be affected by the changes in the API, then you have two options...

  1. make the necessary changes to your code and switch over to version 3.3.0, or...
  2. switch over to version 3.3.0 without changing your code and use this extension to mask the differences so that your code still works.

Of course, if none of the changes between versions 3.2.x and 3.3.0 affect your code, then you can make the switch to 3.3.0 without worries.

Among the things that this patch does,...

  • It defines all of the Music Object 3.2.x extensibility API's global functions, so that versions of Music Object extensions that were written for Music Object 3.2.x can also work with Music Object 3.3.0. If this patch is not used, then it is necessary to use the very latest versions of all Music Object extensions (which have all been updated to work correctly with Music Object 3.3.0).
Getting This Patch

To get the "music-object-p-pre330.js" JavaScript file that implements this patch, simply right-click on this link (click-and-hold for Mac users). Then, from the menu that pops up, select the 'Save Link As...' or 'Save Target As...' option and save the file as "music-object-p-pre330.js" into an appropriate folder of the local copy of your Web site.

In case you have problems saving the file using the above link, the "music-object-p-pre330.js" file can also be found inside the documentation's "javascript" folder, along with the "music-object.js" file.

Dependencies

In order to use this patch, the following JavaScript files must first be sourced in...

  • music-object.js (version 3.3.0 or higher)
