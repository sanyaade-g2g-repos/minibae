Music Object Extension - Instruments (version 1.1.1)
RELEASE DATE: 2001-03-16

What's New In This Version

BUG FIX - Music.getPercussionName, Music.getProgramName, and getProgramName Methods
A bug was found with the Music.getPercussionName and Music.getProgramName static methods and the getProgramName instance method that was causing these methods to return the entire list of instruments in a bank, rather than just the name of the appropriate instrument. This bug has been fixed.
