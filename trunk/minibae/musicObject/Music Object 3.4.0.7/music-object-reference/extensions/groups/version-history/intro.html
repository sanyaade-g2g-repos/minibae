Which Version of This Extension Are You Using?

To find out which version of this extension you are currently using, you should locate the file "music-object-x-groups.js" in the local version of your Web site and then open it in your text editor of choice. Any word processor such as Microsoft Word or Corel WordPerfect will do just fine. Once you have opened up the "music-object-x-groups.js" file, look right at the top of the file where you will find the version number along with credits and copyright information.

How To Use This Version History

Along with the "Introduction" link above, there are links for all the versions of this extension that have a documented version history. If you are upgrading from an earlier version of this extension, you can get an idea of the collective benefits of upgrading by tracking the version histories from the older version that you are currently using, up until the latest version available.
