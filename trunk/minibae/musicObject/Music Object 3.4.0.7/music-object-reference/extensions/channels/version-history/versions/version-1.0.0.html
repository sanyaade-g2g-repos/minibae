Music Object Extension - Channels (version 1.0.0)
RELEASE DATE: 2000-12-28

This is the very first publicly released version of this extension. Beatnik Inc. is excited to be able to expand the Music Object API further through the use of modular extensions that do not increase the code size of the core Music Object API. Our goal is to make your life as a sonified content developer easier, and to help you to create awesome content enhanced by the power of Beatnik's interactive music and audio capabilities.

For more information on Music Object extensions and how they can be used to create more compelling sonified content, consult the section About Music Object Extensions.
