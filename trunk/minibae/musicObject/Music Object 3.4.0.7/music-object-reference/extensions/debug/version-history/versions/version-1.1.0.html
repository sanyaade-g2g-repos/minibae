Music Object Extension - Debug (version 1.1.0)
RELEASE DATE: 2000-12-28

What's New In This Version

UPDATE - Music Object 3.3.0 Compatibility 
Changes have been made in this version of this extension to allow it to work correctly with version 3.3.0 of the core Music Object API. To avoid compatibility problems, this version of this extension should not be used with Music Object versions earlier than 3.3.0.
IMPROVEMENT - Smaller File Size
This version of this extension has been modified as necessary in order to reduce the size of its JavaScript file using Beatnik's proprietary code "scrunching" process (see the section Smaller Size JavaScript Files in the version history entry for Music Object 3.3.0).
WORKAROUND - Bug in Netscape 6

With the initial release of Netscape 6, a problem was discovered that affects error handling. The Debug extension registers a handler for errors by setting the window.onerror property. The function that is registered as the error handler is called each time an error occurs, with parameters being passed to indicate the type of error that occurred, the file that it occurred in, and the specific line number where it occurred.

The bug with Netscape 6 results in useless information being passed to an error handler function, so this kind of error handling is of no value. What's more, if the Debug extension were to catch errors in Netscape 6, then these errors would not get reported to the browser's JavaScript console - one place where useful information is actually displayed. So, as a result of this bug in Netscape 6, it was decided to add some conditional logic to the Debug extension so that an error handler is not registered for this browser. To debug code in Netscape 6, simply pop open the convenient JavaScript console.
