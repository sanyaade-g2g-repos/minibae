The Mixer (explanation)
Description In this example, the Beatnik Player's LiveConnect features are used to allow the user to interact with a file while it is playing, to essentially create a remix of the music composition. The setController and getController methods are used extensively to change the mix as the user adjusts the sliders. There is a slider for each MIDI channel, and the particular aspect of the remix that is controlled by the slider is determined by the MIDI controller that is selected, indicated by the glowing controller name at the top of the mixer. Select a file to play from the drop down list beneath the mixer.

IMPORTANT: To adjust the sliders, first click and release on the slider you want to change. Then mouse up and down over the slider to change its value. You cannot click and drag the slider nob due to basic limitations in JavaScript.
Methods Used getChannelMute, getChannelSolo, magicEmbed, setChannelMute, setChannelSolo, setController, getController, play
Skill Level Advanced
