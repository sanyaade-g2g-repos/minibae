Playing Notes (explanation)
Description Playing notes is one of the key features of the Beatnik player that allows you to sonify Web pages. This example illustrates - in a very straightforward fashion - the various ways one can make elements on an HTML page play musical notes through the Beatnik player.
How It Works The HTML for this example contains several different implementations of note playing. Each section is demarcated in the source code with very visible HTML comments, so you can easily find the source segment for the note playing approach that you would like to use and then build on that code. The notes that are played, throughout all the examples, are based on solfege musical note naming (Do, Re, Mi, So, Fa, La, Ti, Do) and start at middle C. So, the notes that are played are C4, D4, E4, F4, G4, A4, B4, and C5.

We have used the form of the playNote method that can take a MIDI NoteNameSTR parameter. eg. playNote (1,0,0,'G4',100,1000)
