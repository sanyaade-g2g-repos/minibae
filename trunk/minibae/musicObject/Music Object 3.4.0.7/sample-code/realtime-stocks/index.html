Realtime Stocks (explanation)
Description Similar to the JavaScript Seashore example, this example demonstrates how generative music can be used in a very methodal and informing way.
How it Works In this hypothetical scenario, realtime stock information is streamed in and sonified without relying on any precomposed music. The player is instantiated with a tiny stub file and, from there on, all music you hear is generated from scripted algorithms.
Methods Used noteOn, setController, setMonophonic, stubEmbed
Skill Level Advanced
