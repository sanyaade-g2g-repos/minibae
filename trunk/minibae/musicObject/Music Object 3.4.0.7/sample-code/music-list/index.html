Music List (explanation)
Description This example demonstrates how you can instruct a Beatnik Player embedded in an HTML page to play music files from a list. The advantage of this is that the user does not need to leave the preview page, but can keep selecting new files from the list, or read further information on the page as they listen to a file they have chosen. The code is also much easier to author than for a technique that might make use of a frameset.
Methods Used magicEmbed, play
