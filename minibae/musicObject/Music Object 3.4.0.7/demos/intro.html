Introduction

These working demos show the behavior of methods, functions, properties, or features of the Music Object API. The demos include complete explanations, often with links to the various reference pages in the documentation.

How to Navigate

To view the demos, select a desired demo from the drop-down list that appears in the demo area navigation frame above. You can also step through the demos in sequence by using the arrow buttons that surround the listbox. The forward arrow has the effect of moving down the drop-down list, and the back arrow has the effect of moving up the list towards this intro page.
