PART B - step 1: Setting up the HTML

For our example, we will use the exact same layout as the Tiny Little Strips example in the Putting Music On a Page tutorial. Take a look at the sample below and you will see how playing multiple songs at the same time could be a tad bothersome to an end-user.

"Ambient" by Brian Salter
"Classical" arranged by Brian Salter
"Jungle" by Brian Salter
"Lounge" by Brian Salter
"Modern Rock" by Paul Sebastien

Looking at the source HTML for this step, you can see that this is no plain embedding of Beatnik players in a table. Although the layout used for this example is exactly the same as the Tiny Little Strips example, here we have used Music Object's magicEmbed method to embed the players instead of using the plain HTML technique. We have done this in preparation for the next step where we will need to use the Music Object callback methods to register callback handlers for each of the players. Towards the beginning of the source code you will notice that a Music Object instance has been created for each player instance.

<SCRIPT SRC="music-object.js"></SCRIPT>

<SCRIPT LANGUAGE=JavaScript><!-- //

music1 = new Music ();
music2 = new Music ();
music3 = new Music ();
music4 = new Music ();
music5 = new Music ();

// --></SCRIPT>
