STEP 11: Using sonifyFormsWithHandlers to Play Musical Notes

In the previous steps we have seen how both the Music.sonifyFormsWithHandlers and Music.sonifyFormsWithGroovoids methods could be used to achieve exactly the same result. In fact, if all that you wanted to do was sonify all the form elements of a page using Groovoids, then there would be no need to ever use the Music.sonifyFormsWithHandlers. However, as soon as you want to venture beyond using merely Groovoids in your sonification of forms, then the Music.sonifyFormsWithHandlers method is the method for you.

In this example we will show how the Music.sonifyFormsWithHandlers method can be used to sonify form elements using musical notes. Using musical notes opens up a wide new palette of sound textures for sonifying user interface elements - going way beyond what can be done using just the Groovoids that are built into the Beatnik soundbank.

Try out the form below. You will find the sound textures to be noticeably different to those from the previous steps.

In the source code listing above, all the code needed for sonification of the form is highlighted in red. You may wish to refer back to the source code listing for the earlier step, Using the sonifyFormsWithHandlers Method, for a comparison. Instead of specifying handler code to play Groovoids, in this example we are playing musical notes through the Music Object instance "notePlayer".

Towards the bottom of the source code listing you will see the block of code (highlighted in red) that is responsible for actually sonifying the form. The basic approach used in this example is the same as that used in the earlier steps of this tutorial, so we won't discuss things in detail again. In summary, the three key things that are taking place in this block of code are...

  1. A Music Object instance is created, to be used to play musical notes through.
  2. The sonifyForms function is defined, which will be called when the document finishes loading in.
  3. A Beatnik Player instance is embedded for the Music Object instance "notePlayer".
