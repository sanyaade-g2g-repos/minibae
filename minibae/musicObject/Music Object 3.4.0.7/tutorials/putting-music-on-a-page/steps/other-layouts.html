STEP 9: Other Control Panel Layouts

By using different values for the WIDTH, HEIGHT, DISPLAY, MODE, and HIDDEN embedding properties, several different control panel layouts can be achieved. In order to learn more about the specific settings used to achieve these different control panel layouts, consult the appendix Beatnik Player Control Panel Layouts and the reference section Beatnik Player's Embedding Properties.
