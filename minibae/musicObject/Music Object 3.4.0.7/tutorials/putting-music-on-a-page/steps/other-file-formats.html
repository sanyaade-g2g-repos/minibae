STEP 11: Embedding Other File Formats (MIDI, WAV, AIFF, AU)

Although RMF is the premier file format that Beatnik Inc. is promoting for music, the Beatnik Player supports a variety of other music and audio file formats (see the appendix Supported File Types). And it is just as easy to play these file types through the Beatnik Player as it is with RMF files.

In the example below, we are embedding a WAV file that contains some speech...

Take a look at the HTML source code below. The text that is highlighted indicates the filename of the WAV file that was embedded with this Beatnik Player instance. More importantly, notice how the TYPE="audio/rmf" attribute is specifying the MIME type for RMF files. This ensures that the Beatnik Player will play this WAV file, since Beatnik is the only plug-in for Netscape Navigator that handles this MIME type. And even though the MIME type "audio/rmf"suggests that this file may be an RMF file, the Beatnik Player is smart enough to use the file extension (".wav" in this case) for the final determination of the file type.

<EMBED
   SRC="tell-me-about.wav"
   TYPE="audio/rmf"
   WIDTH=144
   HEIGHT=60
   DISPLAY=SONG
   AUTOSTART=FALSE
   LOOP=FALSE
   PLUGINSPAGE="http://www.beatnik.com/to/?player"
>
