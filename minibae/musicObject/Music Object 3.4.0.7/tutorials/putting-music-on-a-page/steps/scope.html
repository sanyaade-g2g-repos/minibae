STEP 3: Scope Out the Music

Called "SCOPE" because it most closely resembles the display of old-style oscilloscopes used by electronics engineers to observe signals, this display mode of the Beatnik Player is arguably the most attractive. While it may not offer the user any information about the author of the composition (as is the case with the SONG display), the dance that it dances is a mesmerizing feast for the eyes. It is an easy way to add a bit of eye candy to an otherwise static page. Better, even, than an animated GIF in that its motion is tied to the music that is currently playing. Livelier music, livelier scope. Mellower music, mellower scope. It kind of captures the mood of the music.

Right-click (click-and-hold for Mac users) anywhere on the Beatnik control panel to the right. From the pop-up menu that appears, select the "Play" menu option to start the music playing and then watch as the audio output is expressed as an animated waveform.

The SCOPE can be specified using the EMBED tag's DISPLAY and MODE attributes. The syntax is as follows:

<EMBED
   SRC="trance.rmf"
   TYPE="audio/rmf"
   WIDTH=144
   HEIGHT=60
   DISPLAY=SYSTEM
   MODE=SCOPE
   AUTOSTART=FALSE
   LOOP=FALSE
   PLUGINSPAGE="http://www.beatnik.com/to/?player"
>

Because you have to put the player into SYSTEM display in order to get the SCOPE mode option, you can no longer have the transport controls in the top portion of the control panel as you would have when SONG display is selected.
