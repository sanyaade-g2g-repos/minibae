Music Object Extension - Song Info (version 1.1.0)
RELEASE DATE: 2001-03-16

What's New In This Version

GENERAL IMPROVEMENT - New Song Info Field Names
The Beatnik Player was updated at version 1.3 to allow the "genre" and "subgenre" info field names to be specified for the showSongInfo parameter when calling the getInfo instance method. The Beatnik Player was updated again at version 2.1 to also allow the "tempo description" and "original source" info field names to be specified. With this maintenance release of the Song Info extension, these newer info field names can now be specified in the SongInfoFieldsSTR parameter when calling the showSongInfo instance method.
