Music Object Extension - Dynamic (version 1.1.0)
RELEASE DATE: 1999-07-27

What's New In This Version

Some exciting new features have been added to this extension since the previous version. Some changes have also been made to allow the extension to work correctly with version 3.2.0 of the core Music Object API.

NEW STATIC METHOD - Music.extractGroup
The new Music.extractGroup static method is useful for pulling out subgroups from an array of string elements. For instance, one could use this method to easily create an array containing only those elements from a source array that are prefixed with "music/ambient/" (ie. all the ambient music files). Find out more...
NEW STATIC METHOD - Music.groovoidGroup
The new Music.groovoidGroup static method is useful for pulling out subgroups of Groovoids from the full list of Groovoids available in the Beatnik Player's soundbank. For instance, one could use this method to easily create an array containing only the user interface Groovoids (prefixed with "UI-"). Find out more...
NEW STATIC METHOD - Music.kwiklistToArray
The new Music.kwiklistToArray static method lets you expand a list that is expressed in the Kwiklist shorthand format into a full array. Among other things, this lets you express large lists of music files in a much condensed form. Find out more...
NEW STATIC METHOD - Music.randomFromGroup
The new Music.randomFromGroup static method lets you easily select random entries from a subgroup of an array of string elements. For instance, one could use this method to easily select a random entry from all the entries in an array that are prefixed with "music/ambient/" (ie. randomly select an ambient music file).
