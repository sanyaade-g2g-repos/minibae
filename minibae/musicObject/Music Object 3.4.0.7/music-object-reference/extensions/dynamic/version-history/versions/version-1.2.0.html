Music Object Extension - Dynamic (version 1.2.0)
RELEASE DATE: 2000-12-28

What's New In This Version

UPDATE - Music Object 3.3.0 Compatibility 
Changes have been made in this version of this extension to allow it to work correctly with version 3.3.0 of the core Music Object API. To avoid compatibility problems, this version of this extension should not be used with Music Object versions earlier than 3.3.0.
IMPROVEMENT - Smaller File Size
This version of this extension has been modified as necessary in order to reduce the size of its JavaScript file using Beatnik's proprietary code "scrunching" process (see the section Smaller Size JavaScript Files in the version history entry for Music Object 3.3.0).
