Music Object Extension - Flash (version 1.1.0)
RELEASE DATE: 1999-07-08

Some exciting new features have been added to this extension since the previous version. Some changes have also been made to allow the extension to work correctly with version 3.2.0 of the core Music Object API.

NEW STATIC METHOD - Music.embedFlashMovie
The new Music.embedFlashMovie static method makes the creation of Beatnik-sonified Flash content even easier by offering a simpler way to embed "wired" Flash movies into a page.
