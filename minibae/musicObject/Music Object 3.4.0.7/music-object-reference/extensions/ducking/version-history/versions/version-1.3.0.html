Music Object Extension - Ducking (version 1.3.0)
RELEASE DATE: 2001-03-02

What's New In This Version

BUG FIX - addDuckingMasters Instance Method
A bug was found with version 1.2 of the Ducking extension that was causing the addDuckingMasters instance method to always fail. This bug has been fixed.
NEW STATIC PROPERTY - Music.duckingAttackTime
The convenient new Music.duckingAttackTime static property is an integer variable, specifying the default value that should be used for the AttackTimeMsINT parameter, if this parameter is not specified when setting up ducking relationships.
NEW STATIC PROPERTY - Music.duckingReleaseTime
The convenient new Music.duckingReleaseTime static property is an integer variable, specifying the default value that should be used for the ReleaseTimeMsINT parameter, if this parameter is not specified when setting up ducking relationships.
NEW STATIC PROPERTY - Music.duckingSustainLevel
The convenient new Music.duckingSustainLevel static property is an integer variable, specifying the default value that should be used for the SustainLevel0to100 parameter, if this parameter is not specified when setting up ducking relationships.
