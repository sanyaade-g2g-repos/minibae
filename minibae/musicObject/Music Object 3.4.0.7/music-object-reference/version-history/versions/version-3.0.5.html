Music Object Version 3.0.5
RELEASE DATE: 1999-03-17

If you are upgrading from Music Object version 2.x and would like to know about the significant changes that have occurred between versions 2.x and 3.0.x, please consult the version history for Music Object Version 3.0.0.

What's New In This Version

NEW STATIC PROPERTY - Music.getPlayerImagesPath
The Music.getPlayerImagesPath has been added to allow content authors to specify a Web location of substitute graphics that are to be displayed in place of the control panel when a user does not have the player installed.
OPTIMIZATIONS
Some general code optimizations have been made since version 3.0.0, which have shaved a little off the "music-object.js" file size as well as making certain processes a little more efficient.
