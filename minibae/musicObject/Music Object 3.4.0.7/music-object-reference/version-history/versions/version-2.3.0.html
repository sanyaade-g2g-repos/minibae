Music Object Version 2.3.0
RELEASE DATE: 1998-05-01

What's New In This Version

 IMPORTANT : Since the Beatnik Player 1.3.0 has been released, you can expect that some end-users may be visiting your site with this version. There are issues with the 1.3.0 player and 4.0x versions of Netscape Navigator. You can only avoid these issues by using version 2.3.0 (or higher) of the Music Object API. If you do not upgrade your content to use at least this version, your content may not work reliably for users with Navigator 4.0x and the 1.3.0 Beatnik Player.
ISSUE WORKAROUND - Callbacks in Navigator 4.0x
The new onPlay, onPause, onStop, and onMetaEvent callbacks that have been introduced in the Beatnik Player 1.3.0 were found to be particularly sensitive to browser issues in Navigator 4.0x versions. In order to reliably make the callbacks work, it was necessary to create a workaround in the Music Object API to prevent the callbacks from being called while a document is still loading. Without this workaround, documents would sometimes freeze or just vanish when close to finished loading. This usually did not result in crashes in the browser, but rendered the pages unusable.
IMPROVEMENT - Debugging Options
It is now easier for authors to debug code with the customizable new Music Object error handler. You can read all about debugging in the Music Object reference section Music Object Debugging Aids.
IMPROVEMENT - General Optimizations
As always, old versions are overhauled and code is optimized for size and performance. Small bugs were found in earlier versions that have now been squashed.
