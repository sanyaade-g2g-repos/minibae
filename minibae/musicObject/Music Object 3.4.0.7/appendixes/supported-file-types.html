Beatnik Player Supported File Types

The Beatnik Player supports all the file types in the table below. If you are serving one of these file types from a Web site, you should be sure to contact your ISP and find out if their server is configured to send the correct MIME type for the appropriate file extension.

 Music Object Note : If you are going to be using the more advanced interactive features of the Beatnik Player, you should be aware that some Music Object methods are not supported (or have no meaning) for some of the player's supported file types. You can find out exactly which Music Object methods apply to the various supported file types by consulting the Music Object Instance Methods section.
