Beatnik Player Control Panel Layouts

The various control panel layouts shown in the table below are possible using different combinations of the WIDTH, HEIGHT, DISPLAY, MODE, and HIDDEN attributes. You can copy-and-paste the HTML code into your own documents and use them as starting points for embedding your own music files. You should also read through the Beatnik Player's Embedding Properties reference section to see what other attributes you might like to include in the EMBED tag.

Differences Between Player Versions

Over the course of the Beatnik Player's history, the control panel graphics have been refined on numerous occasions. Unless you go out of your way in your authoring to specifically require the very latest version of the Beatnik Player (see the section Compatibility Aids: Cutting Edge Content), there is no way of truly guaranteeing how the player's control panel will appear to the user. Some users may have version 2.2 (or higher) installed, while others may have version 2.1, 2.0, or an even older version installed.

Fortunately, the graphics for all versions of the Beatnik Player have been designed to be functionally equivalent to the graphics of earlier versions. So, while different users may see Beatnik Player control panels that are slightly different in appearance, they will all still have the same control panel options available to them. In order to give you a clear idea of the aesthetic differences between the control panel graphics of newer and older Beatnik Player versions, the table below shows the control panel graphics for these different versions in separate columns.
