  • RMF music playback is supported on the WebTV platform, and there is some level of support for the interactive features of the Beatnik Player. Beatnik Inc. is working with Microsoft to extend that level of support in future WebTV products.
  • The Beatnik Player does not currently support the Opera browser.
  • The latest versions of the Beatnik Player and the Music Object API support most customized Netscape browsers distributed by ISPs.
  • The Beatnik Player does not currently support Linux, Sun Solaris, or any other variants on the Unix operating system.
  • The Beatnik Player is not available for Microsoft's Windows 3.1 operating system.
  • On the MacOS, the Beatnik Player only supports PowerPC machines and does not support older 68K machines.
  • As of version 3.4 of the Music Object API, 4.0.x versions of the Netscape Navigator browser are no longer supported.
  • There are certain limitations and authoring considerations to be aware of when creating sonified content that is to be fully compatible with Internet Explorer on the Mac OS. For more information, see the section Issues for Internet Explorer on the MacOS.
