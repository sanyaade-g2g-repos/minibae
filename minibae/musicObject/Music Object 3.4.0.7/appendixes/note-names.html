Note Names and Numbers

Shown in the table below are all the MIDI note numbers from 0 to 127. They are divided up into columns for each octave, with 12 notes in each column. The left hand side of each octave column contains the note names, and the right hand side contains the corresponding MIDI note numbers. The note names and note numbers as listed in this table can be used with the Music Object's playNote, noteOn, and noteOff instance methods, as well as the getNoteName and getNoteNumber static methods.
